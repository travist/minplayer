Class: display

minplayer. display

Base class used to provide the display and options for any component deriving from this class. Components who derive are expected to provide the elements that they define by implementing the getElements method.

new display(name, context, options, queue)

Parameters:
Name Type Description
name string The name of this plugin.
context object The jQuery context this component resides.
options object The options for this component.
queue object The event queue to pass events around.
